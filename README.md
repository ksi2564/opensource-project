# README

## 마켓컬리 클론 코딩을 진행하였음.
### 사용한 것
- ruby on rails
- bootstrap
- sqlite3
---
### 주요 기능
+ admin 페이지 커스터마이징
+ 상품 공개 유무 기능
+ 주문 및 결제 기능
  + daum API 사용 주소 검색 기능
  + IAMPort API 사용 결제 기능
  + 총 결제금액 40,000원 이하 배송비 청구
  + 환불 기능까지 구현
 ...
